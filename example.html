If the first line of the file starts with # jemdoc, special functions like menus will be used.

Example

Here are some text features. I could link somewhere or insert a raw link to another page like download.html. I could use monospace, too. I could write special characters like #, $ and + by just using a backslash (\) in front of those characters. Or automatically detect an email address.

Save the file as index.jemdoc, say, and simply call jemdoc index (after downloading jemdoc, of course).
